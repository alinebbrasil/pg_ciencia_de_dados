# Repositório de MVPs — Pós-Graduação em Ciência de Dados e Analytics (PUC-Rio)

Este repositório é destinado ao **armazenamento dos MVPs desenvolvidos ao longo da Pós-Graduação em Ciência de Dados e Analytics da PUC-Rio**.

Aqui estão organizadas as pastas contendo os arquivos e projetos referentes a **cada período letivo**, permitindo o acompanhamento da evolução dos trabalhos acadêmicos desenvolvidos durante o curso.
